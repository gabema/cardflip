# CardFlip

# Rationale
Create a web only app that can be hosted via github pages developed
in C# using browser technologies.

## Development

### Manipulating cards via Dev Console


```es6
localStorage.setItem('wordPairs', JSON.stringify([
    { Front: 'Hello', Back: 'World' },
    { Front: 'Foo', Back: 'Bar' }
]));
```
